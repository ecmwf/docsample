# My Python Project

A project template containing a library, CLI, Jupyter Notebook, and Sphinx documentation.

## Installation

```bash
pip install .
```

## Usage

### Library

```python
from my_python_project.core import main_feature
print(main_feature("Hello"))
```

### CLI

```bash
my-cli Hello
```

### Notebook

Open `notebooks/example_notebook.ipynb`.
